10: # Cell-Cell Communication Analysis (CellChat v2)
11: 
12: ## When to Use This Skill
13: 
14: ✅ **Use when:**
15: - You have an annotated scRNA-seq dataset (Seurat object with cell type labels)
16: - You want to identify ligand-receptor interactions between cell types
17: - You want to visualize communication networks (chord diagrams, bubble plots)
18: - You want to find dominant sender/receiver cell populations
19: - **Chains from** `scrnaseq-seurat-core-analysis` output (`seurat_processed.rds`)
20: 
21: ❌ **Don't use when:**
22: - Data is not annotated (run `scrnaseq-seurat-core-analysis` first)
23: - You need spatial cell-cell communication (CellChat v2 supports this but requires spatial coordinates)
24: - You want gene regulatory networks (use `grn-pyscenic` instead)
25: - You have bulk RNA-seq data

44: ## Inputs
45: 
46: **Required:**
47: - **Seurat object (.rds)** with:
48:   - Normalized expression data (`@assays$RNA@data`)
49:   - Cell type annotations in metadata (e.g., `celltype` column)
50:   - Minimum 3 cell types, ≥10 cells per type recommended
51: 
52: **Accepted sources:**
53: - `seurat_processed.rds` from `scrnaseq-seurat-core-analysis` (chains directly)
54: - Any annotated Seurat v5 object
55: - Example PBMC data (auto-loaded if no file provided)
56: 
57: ## Outputs
58: 
59: **CSV tables:**
60: - `significant_interactions.csv` — All significant L-R pairs with source, target, pathway, probability
61: - `pathway_summary.csv` — Pathway-level communication summary
62: - `interaction_count_matrix.csv` — Cell type × cell type interaction counts
63: - `interaction_strength_matrix.csv` — Cell type × cell type communication weights
64: - `signaling_roles.csv` — Centrality scores (sender, receiver, mediator, influencer per pathway)
65: - `top_interactions.csv` — Top 20 interactions ranked by probability
66: 
67: **Visualizations (PNG + SVG):**
68: - `interaction_count_network` — Circle plot of interaction counts
69: - `interaction_strength_network` — Circle plot of communication strength
70: - `chord_aggregated` — Chord diagram of the full communication network
71: - `bubble_ligand_receptor` — Bubble plot of L-R pairs by cell type pairs
72: - `signaling_outgoing_heatmap` — Outgoing signaling patterns by cell type
73: - `signaling_incoming_heatmap` — Incoming signaling patterns by cell type
74: - `signaling_role_scatter` — Dominant senders vs receivers scatter
75: 
76: **Analysis objects (RDS):**
77: - `cellchat_object.rds` — Complete CellChat object for downstream use
78:   - Load with: `cellchat <- readRDS('cellchat_object.rds')`
79:   - Required for: multi-condition comparison, pathway-specific deep dives
80: 
81: **Reports:**
82: - `analysis_report.md` — Markdown report (always generated)
83: - `analysis_report.pdf` — PDF report (requires rmarkdown + LaTeX)

171: ## Common Issues
172: 
173: | Issue | Cause | Solution |
174: |-------|-------|----------|
175: | **CellChat not found** | Not installed from GitHub | `devtools::install_github("jinworks/CellChat")` — must use **jinworks** repo (not sqjin) |
176: | **"presto" required for Wilcoxon test** | Missing presto package | `remotes::install_github('immunogenomics/presto')` — script falls back to standard test if unavailable |
177: | **No significant interactions** | Too few cells per type or stringent filtering | Lower `min.cells` parameter or merge rare cell types |
178: | **Memory error on large datasets** | >50k cells uses substantial RAM | Subsample or increase memory; see [references/cellchat-guide.md](references/cellchat-guide.md) |
179: | **Chord diagram error** | Missing circlize package | `install.packages('circlize')` |
180: | **SVG export error "svglite required"** | Missing optional dependency | Use `generate_all_plots()` — it handles fallback automatically. DO NOT try to install svglite manually. |
181: | **svglite dependency conflict** | System library version mismatch | Normal — `generate_all_plots()` falls back to base R svg() device automatically. Both PNG and SVG will be created. |
182: | **"group.by not found"** | Wrong column name for cell types | Check: `colnames(seurat_obj@meta.data)` |
183: | **Seurat v5 slot error ("no slot of name images")** | Old Seurat object from v3/v4 | Script handles this — `UpdateSeuratObject()` is called automatically |
184: | **NMF not available** | NMF package not installed | `install.packages('NMF')` |
185: | **PDF report skipped** | No LaTeX installation | `install.packages('tinytex'); tinytex::install_tinytex()` — markdown report still available |
