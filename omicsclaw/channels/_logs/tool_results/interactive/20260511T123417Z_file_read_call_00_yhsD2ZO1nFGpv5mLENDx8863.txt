243: ## Common Issues
244: 
245: | Issue | Cause | Solution |
246: |-------|-------|----------|
247: | `ImportError: No module named 'scanpy'` | Not installed | `pip install scanpy anndata numpy pandas matplotlib seaborn` |
248: | Low cell retention (<70%) | Strict QC thresholds | Use MAD (nmads=5→7) or tissue-specific thresholds |
249: | Out of memory | Large dataset (>50k cells) | Use backed mode or subsample |
250: | Clusters driven by batch | Insufficient integration | Use scVI, increase n_latent, check confounding |
251: | Poor UMAP separation | Wrong parameters | Check PCA elbow, use 20-40 PCs, adjust n_neighbors |
252: | High MT% in all cells | Degradation or tissue-specific | Check distribution — bimodal: stricter filter; uniform: may be biological |
253: | `FileNotFoundError: barcodes.tsv.gz` | Wrong directory | Verify 10X output files present. Use `import_h5_data()` for .h5 |
254: | H5AD export hangs or is very slow | Large file write with compression | Normal for >50 MB files. Script uses fast `lzf` compression. If still slow, pass `compression=None` to `save_h5ad()`. |
255: | `NameError` after export interruption | Kernel restart lost variables | Re-run from Step 1 to restore `adata`. Export is idempotent — safe to re-run. |
256: 
257: **Expected warnings (not errors):**
258: 
259: | Warning | Meaning | Action |
260: |---------|---------|--------|
261: | SVG export failed | Optional SVG dependency unavailable | Normal — PNG always generated. Both created in most environments. |
262: | Detected doublet rate differs from expected | Scrublet threshold or pre-filtering | Inspect `adata.obs['doublet_score'].hist()`. Adjust threshold if needed. |
263: | Pseudobulk DE blocked: N=1 | Insufficient replicates | Need ≥2 per condition. Use cell-level Wilcoxon for exploratory only. |
264: | Batch-condition confounding | Condition has 1 sample | Clustering valid. Composition comparisons need caveats. |
265: | CellTypist labels suspect (ILC, HSC) | Automated misclassification | Cross-check markers. Relabel ILC→NK or HSC→Unassigned. |
266: | Multimodal data detected | RNA-only workflow on CITE-seq | Note in reports: "RNA modality only; ADT not analyzed." |
