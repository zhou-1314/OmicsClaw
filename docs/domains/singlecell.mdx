## 本领域解决什么问题

单细胞组学域同时覆盖 scRNA-seq 与 scATAC-seq，提供从原始 FASTQ 到生物学解读的完整流程，包括计数、QC、过滤、归一化、聚类、细胞类型注释、差异表达、批次整合、轨迹推断、RNA velocity 与基因调控网络。与空间域的核心区别在于：单细胞域无空间坐标，专注细胞身份、状态转变与调控机制。

## 典型分析流程 ASCII 图

```text
FASTQ
   │
   ▼
sc-fastq-qc           ← 原始读长质控 (FastQC / MultiQC)
   │
   ▼
sc-count              ← cellranger / STARsolo / kallisto → AnnData
   │
   ▼
sc-standardize-input  ← 规范化外部 h5ad 合约
   │
   ▼
sc-qc                 ← QC 指标（线粒体 / 核糖体百分比）
   │
   ▼
sc-ambient-removal    ← 去除环境 RNA 污染
   │
   ▼
sc-doublet-detection  ← 双细胞检测
   │
   ▼
sc-filter             ← 细胞 / 基因过滤
   │
   ▼
sc-preprocessing      ← 归一化 + HVG + PCA
   │
   ▼
sc-clustering         ← 邻居图 + UMAP + Leiden 聚类
   │
   ├─→ sc-cell-annotation    ← 细胞类型注释 (marker / CellTypist / SingleR)
   ├─→ sc-markers            ← 聚类 marker 基因排名
   ├─→ sc-de                 ← 差异表达 (Wilcoxon / MAST / pseudobulk)
   ├─→ sc-batch-integration  ← 多样本批次整合 (Harmony / scVI)
   ├─→ sc-enrichment         ← 通路富集 (ORA / GSEA)
   ├─→ sc-pseudotime         ← 拟时序 (DPT / Palantir / CellRank)
   ├─→ sc-velocity           ← RNA velocity (scVelo)
   ├─→ sc-grn                ← 基因调控网络 (pySCENIC)
   └─→ sc-cell-communication ← 细胞通讯 (LIANA / CellChat)
```

## 何时不适用本领域

- **数据带空间坐标**（Visium / Xenium / MERFISH）：用 [spatial 域](/docs/domains/spatial)
- **bulk RNA-seq 数据**：用 [bulkrna 域](/docs/domains/bulkrna)
- **想做 scDNA-seq / scBS-seq（甲基化）**：当前 skill 不覆盖
- **CITE-seq 蛋白通道**：部分 skill 支持，但不是主要场景
