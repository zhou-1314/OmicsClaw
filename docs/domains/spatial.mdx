## 本领域解决什么问题

空间转录组域覆盖从原始数据加载到高级下游分析的完整流程，包括 QC、域识别、SVG 挖掘、解卷积、细胞通讯、轨迹与 CNV 推断。与单细胞域的核心区别在于：空间转录组保留每个 spot 或细胞的二维坐标，分析的核心问题是"基因表达在哪里"——不仅描述细胞类型，还揭示其空间分布、邻域关系与组织微环境结构。

## 典型分析流程 ASCII 图

```text
原始数据 (h5ad / 10x output)
   │
   ▼
spatial-preprocess     ← QC + 标准化 + HVG + PCA + UMAP
   │
   ▼
spatial-domains        ← 空间域识别 (leiden / SpaGCN / ...)
   │
   ├─→ spatial-de              ← 各域 marker 基因
   ├─→ spatial-genes           ← 空间可变基因 (SVG)
   ├─→ spatial-deconv          ← 细胞类型解卷积
   ├─→ spatial-communication   ← 配体-受体通讯
   ├─→ spatial-trajectory      ← 拟时序
   └─→ spatial-cnv             ← 拷贝数
```

## 何时不适用本领域

- **数据没有空间坐标**（如普通 scRNA-seq）：建议使用 [singlecell 域](/docs/domains/singlecell)
- **bulk RNA-seq 数据**：建议使用 [bulkrna 域](/docs/domains/bulkrna)
- **想做组织切片图像分析（H&E 细胞分割）**：当前 skill 不覆盖图像深度学习，需借助外部工具（如 StarDist、Cellpose）
- **DNA 层面的空间分析（spatial DNA-seq）**：不在本域范围内
