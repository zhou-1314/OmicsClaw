Implement two functions:

**plot_embedding_discrete(data_dir, out_path, params)**

Produces a CellDimPlot-equivalent: discrete categorical scatter with cluster labels.

Logic:
1. Read `annotation_embedding_points.csv` from data_dir.
2. Determine color column: `params[["color_by"]]` if provided, else use `"cell_type"` if present, else first non-coordinate column.
3. Convert color column to factor (preserves legend order).
4. Count unique levels `n_groups`. Call `omics_palette(n_groups)` for fill colors.
5. Build ggplot:
   - `geom_point(size=0.6, alpha=0.7)` — small points for dense cell clouds
   - `scale_color_manual(values = omics_palette(n_groups))` — categorical palette
   - Compute per-group centroid (mean dim1, mean dim2) → `geom_text(aes(label=group), data=centroids, size=3.5, fontface="bold", check_overlap=TRUE)`
   - `labs(x = "UMAP 1", y = "UMAP 2", color = color_col, title = paste0("Cell embedding — ", color_col))`
   - `theme_omics()` + `theme(legend.key.size = unit(0.4, "cm"))` for compact legend
   - `guides(color = guide_legend(override.aes = list(size = 3, alpha = 1)))` — readable legend dots
6. `ggsave_standard(p, out_path, width = 8, height = 6)`

**plot_embedding_feature(data_dir, out_path, params)**

Produces a FeatureDimPlot-equivalent: continuous value overlay with viridis.

Logic:
1. Read `annotation_embedding_points.csv` from data_dir.
2. Determine feature column: `params[["feature"]]` if provided, else `"annotation_score"` if present in df, else stop with informative error message ("No continuous feature column found — pass feature=<colname>").
3. Convert feature column to numeric; warn and skip NA rows.
4. Build ggplot:
   - `geom_point(size=0.6, alpha=0.8)` colored by feature value
   - `scale_color_viridis_c(option="magma", name=feature_col, na.value="#CCCCCC")` — viridis continuous, NA cells grey
   - `labs(x="UMAP 1", y="UMAP 2", title=paste0(feature_col, " — embedding overlay"))`
   - `theme_omics()`
5. `ggsave_standard(p, out_path, width = 8, height = 6)`

Both functions must be wrapped in tryCatch with `cat("ERROR:", ..., file=stderr()); quit(status=1)` in the error handler — matching the pattern documented at the bottom of common.R.

from pathlib import Path

# Generate synthetic annotation_embedding_points.csv
import pandas as pd, numpy as np, os
tmpdir = Path(tempfile.mkdtemp())
fig_dir = tmpdir / 'figure_data'
fig_dir.mkdir()
np.random.seed(42)
n = 200
df = pd.DataFrame({
    'cell_id': [f'cell_{i}' for i in range(n)],
    'dim1': np.random.randn(n),
    'dim2': np.random.randn(n),
    'cell_type': np.random.choice(['T cell', 'B cell', 'NK cell', 'Monocyte'], n),
    'annotation_score': np.random.uniform(0, 1, n),
})
df.to_csv(fig_dir / 'annotation_embedding_points.csv', index=False)

out_discrete = tmpdir / 'r_embedding_discrete.png'
out_feature  = tmpdir / 'r_embedding_feature.png'

r_script = Path('skills/singlecell/_lib/viz/r/registry.R')
env = os.environ.copy()
env['R_LIBS'] = '/data2/liying/OMICSCLAW_R_LIBS'

for renderer, out in [
    ('plot_embedding_discrete', out_discrete),
    ('plot_embedding_feature',  out_feature),
]:
    result = subprocess.run(
        ['Rscript', str(r_script), renderer, str(fig_dir), str(out)],
        capture_output=True, text=True, env=env
    )
    if result.returncode != 0:
        print(f'FAIL {renderer}:', result.stderr[-500:])
        sys.exit(1)
    if not out.exists() or out.stat().st_size < 5000:
        print(f'FAIL {renderer}: PNG missing or too small ({out.stat().st_size if out.exists() else 0} bytes)')
        sys.exit(1)
    print(f'PASS {renderer}: {out.stat().st_size} bytes')
print('Both renderers passed.')
