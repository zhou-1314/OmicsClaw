**plot_marker_heatmap(data_dir, out_path, params)**

Produces a GroupHeatmap equivalent: marker genes × clusters heatmap with blue-white-red
diverging color scale and per-cluster column slicing.

Logic:
1. Read `markers_top.csv` from data_dir.
   - If file does not exist or has 0 rows: `cat("ERROR: markers_top.csv not found or empty\n", file=stderr()); quit(status=1)`
2. Determine value column: if `logfoldchanges` column exists and is not all-NA, use it.
   Otherwise fall back to `scores`. Store as `val_col`.
3. Remove rows where the value column is NA.
4. Pivot to wide matrix: `pivot_wider(df, id_cols="names", names_from="group", values_from=val_col, values_fill=0, values_fn=mean)`. Convert to plain data frame, set rownames to `names` column, drop `names` column, convert remaining columns to numeric matrix.
5. Limit to top N genes per cluster to keep the plot readable:
   - `n_top <- as.integer(params[["n_top"]] %||% 10)` where `%||%` is `function(a,b) if(!is.null(a)) a else b`
   - Re-read df, `group_by(group) |> slice_max(order_by=.data[[val_col]], n=n_top, with_ties=FALSE)` then pivot as above. This ensures genes are ordered within clusters.
6. Build column split factor: `col_split <- factor(colnames(mat), levels=colnames(mat))`
7. Build color function: `col_fun <- colorRamp2(c(min(mat), 0, max(mat)), c("#2166AC", "white", "#D6604D"))` — blue-white-red diverging.
8. Build ComplexHeatmap:
   ```r
   ht <- Heatmap(
     mat,
     name            = val_col,
     col             = col_fun,
     column_split    = col_split,
     cluster_rows    = FALSE,
     cluster_columns = FALSE,
     cluster_row_slices    = FALSE,
     cluster_column_slices = FALSE,
     show_row_names  = TRUE,
     show_column_names = FALSE,
     row_names_side  = "left",
     row_names_gp    = gpar(fontsize = 8),
     column_title_gp = gpar(fontsize = 9, fontface = "bold"),
     border          = TRUE,
     heatmap_legend_param = list(
       title          = val_col,
       legend_direction = "horizontal",
       title_position = "topcenter"
     )
   )
   ```
9. Save via `png(out_path, width=10, height=max(4, nrow(mat)*0.18), units="in", res=200); draw(ht, heatmap_legend_side="bottom"); dev.off()`. Note: ComplexHeatmap uses base R graphics, NOT ggplot2 — do NOT use `ggsave_standard()` here. After `dev.off()`, print `cat("Saved:", out_path, "\n")` to match the ggsave_standard convention.

The full tryCatch wrapper applies to the entire function body:
```r
plot_marker_heatmap <- function(data_dir, out_path, params) {
  tryCatch({
    suppressPackageStartupMessages({
      library(ComplexHeatmap)
      library(circlize)
      library(tidyr)
    })
    ...
  }, error = function(e) {
    cat("ERROR:", conditionMessage(e), "\n", file = stderr())
    quit(status = 1)
  })
}
```

            'names': gene,
            'scores': np.random.uniform(5, 25),
            'logfoldchanges': np.random.uniform(0.5, 3.5),
            'pvals': np.random.uniform(1e-10, 0.05),

df = pd.DataFrame(rows)
