Create `omicsclaw/r_scripts/sc_gsea_r.R`. This R script receives a ranked DE/marker gene
CSV from Python, runs clusterProfiler GSEA for each group in the table, and writes one
result CSV.

**CLI interface:**
```
Rscript sc_gsea_r.R <de_csv> <output_dir> <species> <db> <score_type>
```
- `de_csv`: CSV with columns `gene`, `avg_log2FC`, `group` (group column name may vary — detect it)
- `output_dir`: directory where `gsea_r_results.csv` will be written
- `species`: `"Homo_sapiens"` or `"Mus_musculus"` (default `"Homo_sapiens"`)
- `db`: `"GO_BP"`, `"KEGG"`, or `"Reactome"` (default `"GO_BP"`)
- `score_type`: `"std"`, `"pos"`, `"neg"` (default `"std"`)

**Implementation steps:**

1. Parse CLI args with defaults for args 3-5 when not provided.

2. Load libraries (suppressPackageStartupMessages):
   `clusterProfiler`, `DOSE`, `org.Hs.eg.db` (if human) or `org.Mm.eg.db` (if mouse),
   `dplyr`, `GO.db`

3. Read the DE CSV. Detect the group column: prefer `group1` > `group` > `cluster` > first
   non-numeric, non-gene column. Normalize gene names to uppercase (for human consistency).

4. Remove rows where `avg_log2FC` is NA or infinite.

5. Auto-detect `scoreType`: if all log2FC > 0 → force "pos"; if all < 0 → force "neg"; else use arg.

6. For each unique group in the table:
   a. Filter rows for that group.
   b. Build ranked named vector: `sort(setNames(df$avg_log2FC, df$gene), decreasing=TRUE)`.
      Deduplicate by keeping the entry with max abs score per gene name.
   c. Skip group if fewer than 10 genes remain after deduplication.
   d. Fetch gene sets from `org.Hs.eg.db` / `org.Mm.eg.db` for the requested db:
      - GO_BP: use `clusterProfiler::enrichGO` gene set table, or build TERM2GENE from GO.db
      - KEGG: `clusterProfiler::download_KEGG` (or `msigdbr` if available) — for simplicity,
        use `clusterProfiler::gseKEGG` directly (converts gene symbols internally)
      - Reactome: skip if `ReactomePA` not installed; fall back to GO_BP with a warning
   e. Run GSEA:
      ```r
      tryCatch({
        res <- clusterProfiler::GSEA(
          geneList  = ranks,
          TERM2GENE = term2gene,
          minGSSize = 10,
          maxGSSize = 500,
          pvalueCutoff = 1.0,
          scoreType = score_type,
          by = "fgsea",
          verbose = FALSE
        )
        result_df <- as.data.frame(res)
        result_df$Group    <- group_name
        result_df$Database <- db
        all_results[[group_name]] <- result_df
      }, error=function(e) {
        cat("WARNING: GSEA failed for group", group_name, ":", conditionMessage(e), "\n",
            file=stderr())
      })
      ```

7. Combine all group results into one data.frame. Rename columns for Python:
   Keep: ID, Description, NES, pvalue, p.adjust, core_enrichment, Group, Database.
   If `setSize` column exists, keep it.

8. Write `gsea_r_results.csv` to output_dir. If no results at all (empty combined df),
   write an empty CSV with the correct headers and exit 0 (not 1 — empty is not an error).

9. Wrap the entire outer block in `tryCatch(..., error=function(e) { cat(...); quit(status=1) })`.

```python
def _run_gsea_r(adata, ranking_df, output_dir, params):
    """Run clusterProfiler GSEA via R bridge. Returns summary dict."""
    import warnings
    from omicsclaw.core.r_script_runner import RScriptRunner, RScriptError

    r_script = R_SCRIPTS_DIR / "sc_gsea_r.R"
    if not r_script.exists():
        raise FileNotFoundError(f"R script not found: {r_script}")

    # Export ranking table for R
    r_work_dir = output_dir / "r_work"
    r_work_dir.mkdir(parents=True, exist_ok=True)
    de_csv = r_work_dir / "de_for_gsea_r.csv"
    ranking_df.to_csv(de_csv, index=False)

    species   = params.get("species", "Homo_sapiens")
    db        = params.get("db", "GO_BP")
    score_type = params.get("score_type", "std")

    runner = RScriptRunner(timeout=600)
    result_csv = r_work_dir / "gsea_r_results.csv"
    r_success = True
    try:
        runner.run_script(
            r_script,
            args=[str(de_csv), str(r_work_dir), species, db, score_type],
            expected_outputs=["gsea_r_results.csv"],
            output_dir=r_work_dir,
        )
    except RScriptError as exc:
        warnings.warn(f"gsea_r R script failed (Python figures unaffected): {exc}")
        r_success = False

    enrichment_df = pd.DataFrame()
    if r_success and result_csv.exists():
        try:
            enrichment_df = pd.read_csv(result_csv)
        except Exception as exc:
            warnings.warn(f"Could not read gsea_r results CSV: {exc}")

    # Merge to adata.uns
    if not enrichment_df.empty:
        adata.uns["gsea_r_results"] = enrichment_df.to_dict(orient="list")

    return {
        "method": "gsea_r",
        "r_success": r_success,
        "n_terms": len(enrichment_df),
        "species": species,
        "db": db,
    }
```
